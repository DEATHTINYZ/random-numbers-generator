# random-numbers-generator
- ให้เขียนโปรแกรม  
- Random เลขจำนวนนับ ทั้งหมด 10,000 ตัวเลขบันทึกลงในไฟล์ที่ชื่อ Input1000.txt
- ใช้ Bubble Sort เรียงข้อมูล 
- โดยอ่านข้อมูลจากไฟล์ Input1000.txt และบันทึกคำตอบลงในไฟล์ Output10000.txt
